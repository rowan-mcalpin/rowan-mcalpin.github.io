fun calculateFromDriveTurnStrafe(drive: Double, turn: Double, strafe: Double, regulate: Boolean = true)

Given a drive speed (-1 to 1), turn speed (-1 to 1), and strafe speed (-1 to 1), set motor powers accordingly. If set to regulate, then it won't attempt to set the motor powers higher than 1.0, which would result in clipping the value instead of maintaining the ratio (which is usually the desired action).
